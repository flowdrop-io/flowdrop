This guide walks you through creating your first workflow with FlowDrop.

## Creating a Workflow

Workflows in FlowDrop are composed of nodes connected by edges. Each node represents a step in your workflow, and edges define the flow between steps.

## Node Types

FlowDrop comes with several built-in node types. You can also register custom node types using the plugin system.

## Importing and Exporting

FlowDrop supports multiple workflow formats:

- **FlowDrop native** — the default JSON format
- **Agent Spec (Oracle)** — Oracle Open Agent Spec format for AI agent workflows
