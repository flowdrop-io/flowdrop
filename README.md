# FlowDrop

Build beautiful workflow editors in minutes, not months.

## Repository Structure

| Directory | Description |
|-----------|-------------|
| [libs/flowdrop](libs/flowdrop/) | The FlowDrop library ([npm](https://www.npmjs.com/package/@d34dman/flowdrop)) |
| [apps/docs](apps/docs/) | Documentation site |
| [apps/api-docs](apps/api-docs/) | OpenAPI documentation (Redocly) |
| [apps/example-svelte](apps/example-svelte/) | Svelte/SvelteKit examples |

## Getting Started

Each project is independent with its own dependencies. To work on the library:

```bash
cd libs/flowdrop
pnpm install
pnpm dev
```
